# 🚚 화물 운송 시스템 (Freight Logistics System)

화물 운송 과정을 디지털화하고, 고객부터 운전자까지 효율적으로 연결하는 **화물 운송 관리 시스템**입니다.  
고객의 오더 등록부터 물류센터의 배차, 운송 추적까지 전 과정을 관리할 수 있습니다.

---

## 📌 주요 기능 (Features)

### 👤 고객 (Customer)
- 화물 운송 요청(오더) 등록
- 오더 상태 확인 (배차/운송/완료)
- 오더 취소 및 수정

### 🏢 물류센터 (Logistics Center)
- 오더 확인 및 배차 처리
- 운송 스케줄 관리
- 차량 및 운전자 배정

### 🚛 차량주인 (Vehicle Owner)
- 차량 등록 및 관리
- 운전자 배정
- 오더 수락 여부 결정

### 👨‍✈️ 운전자 (Driver)
- 운송 상태 업데이트 (출발, 도착 등)
- 위치 정보 제공
- 문제 상황 보고

---
